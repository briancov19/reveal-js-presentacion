# Eficiencia energética UCU

Presentación académica en Reveal.js sobre el avance del proyecto del Grupo N.º 1, Equipo 6.

## Requisitos

- Node.js 20 o superior.
- npm.

## Instalación

```bash
npm install
```

## Ejecutar en modo desarrollo

```bash
npm run dev
```

Luego abrir la URL local que muestra la terminal, normalmente:

```text
http://127.0.0.1:8000/
```

## Generar versión final

```bash
npm run build
```

La versión construida queda en la carpeta `site`.

## Estructura principal

```text
index.html
src/styles.css
src/main.js
package.json
README.md
```

## Imágenes opcionales

La presentación no depende de imágenes externas. Si se agregan estos archivos, se muestran automáticamente en las diapositivas correspondientes:

```text
assets/prototipo-bosquejo.png
assets/tinkercad-avance.png
assets/matriz-evaluacion.png
```

Si no están disponibles, se usan ilustraciones creadas con HTML y CSS.
